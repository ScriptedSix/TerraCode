# Typography Guide

Font Family: Inter

## Headings
H1:
- Font: Inter
- Weight: Bold
- Size: 34px
- Line height: Auto
- Letter spacing: 0%

H2:
- Font: Inter
- Weight: Bold or Medium
- Size: 24px
- Line height: Auto

## Body Text
- Font: Inter
- Weight: Regular
- Size: 14px or 16px 
- Line height: Auto

## Button Text
- Font: Inter
- Weight: SemiBold / Bold
- Size: 20px
